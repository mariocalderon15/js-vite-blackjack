Blackjack Game

A complete implementation of the classic casino card game Blackjack.
This project was built as part of my programming portfolio to demonstrate problem-solving skills, game logic implementation, and clean code organization.

Project Overview

Blackjack is a card game where the objective is to beat the dealer by having a hand value closer to 21 without exceeding it. This project recreates the core mechanics of the game, including card dealing, player decisions, dealer logic, and win/loss conditions.

Features:

Random card generation
Player actions (Hit / Stand)
Dealer automated decision logic
Score calculation
Win / Lose / Draw detection
Clean and modular code structure
